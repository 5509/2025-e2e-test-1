このデモについて

このデモアプリケーションは、Playwrightを使ったE2Eテストの学習用に作成されました。

目的

  • シンプルなWebページでのE2Eテストの基本を学ぶ
  • JavaScriptエラーの検出テストを実装する
  • GitHub Actions CIでの自動テストを体験する

技術スタック

  • HTML5
  • CSS3
  • Vanilla JavaScript
  • Playwright
  • GitHub Actions

概要

このデモでは、基本的なタブ切り替え機能を持つ3つのページを用意しています。

特徴

各ページで共通のJavaScriptファイルを使用し、一貫したUI操作を提供します。

使い方

ナビゲーションメニューから各ページに移動し、タブ機能を試してみてください。
